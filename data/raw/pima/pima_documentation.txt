PIMA Indians Diabetes Dataset — Documentation
==============================================

Source: Kaggle / UCI Machine Learning Repository
URL: https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database

Dataset Info:
- Rows: 768
- Columns: 9 (8 features + 1 target)
- Task: Binary Classification (Diabetes / No Diabetes)

Features:
1. Pregnancies              — عدد مرات الحمل
2. Glucose                  — مستوى الجلوكوز في الدم (mg/dL)
3. BloodPressure            — ضغط الدم الانبساطي (mm Hg)
4. SkinThickness            — سمك طية الجلد (mm)
5. Insulin                  — مستوى الإنسولين (mu U/ml)
6. BMI                      — مؤشر كتلة الجسم (kg/m²)
7. DiabetesPedigreeFunction — دالة الوراثة للسكري
8. Age                      — العمر (سنة)

Target:
- Outcome: 0 = No Diabetes | 1 = Diabetes

Class Distribution:
- No Diabetes (0): 500 (65.1%)
- Diabetes    (1): 268 (34.9%)

Hidden Missing Values (Zeros):
- Glucose:        5  (0.7%)
- BloodPressure:  35 (4.6%)
- SkinThickness:  227 (29.6%)
- Insulin:        374 (48.7%)  ← الأخطر
- BMI:            11 (1.4%)

Preprocessing Applied:
1. Replace zeros with NaN in medical columns
2. KNNImputer (n_neighbors=5) for missing values
3. SMOTE on Train only (186 synthetic samples)
4. StandardScaler for feature scaling

Model Results:
- Accuracy:  74.7%
- F1-Score:  68.3%
- ROC-AUC:   83.5%
- CV F1:     80.4% +/- 3.7%
- CV AUC:    88.4% +/- 1.8%

Top 5 Features:
1. Insulin                  (26.4%)
2. Glucose                  (24.7%)
3. BMI                      (14.6%)
4. Age                      (10.4%)
5. DiabetesPedigreeFunction (7.3%)

Files:
- model_int8.onnx  — XGBoost model exported to ONNX (21.1 KB)
- features.json    — Feature names + metrics + scaler params
- imputer.pkl      — KNNImputer fitted on training data
- scaler.pkl       — StandardScaler fitted on training data

Citation:
